# Projects Marketplace

This repository hosts a Claude Code plugin marketplace for Projects product domains.

Current marketplace id:

- `projects`

Current plugins:

- `clinic-operations` — Dr Souphi bookings, workflow recovery, and booking configuration

## Install

Add the marketplace:

```bash
claude plugin marketplace add mafifi/claude-marketplace
```

Install the Clinic Operations plugin:

```bash
claude plugin install clinic-operations@projects
```
